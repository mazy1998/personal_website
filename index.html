SPLATROGRAPH

I started Splatrograph when it became evident that no simple API existed to create 3D models from 2D images. Akin to LLM APIs like ChatGPT and Claude, I wanted to create an API that developers could use to generate 3D models without specialized knowledge. The goal is to make an open-source API that can be used to run 3D reconstruction pipelines with the ability to semantically analyze them as well.

By making 3D model generation as accessible as text generation, I hope to empower developers to build applications in fields like visualization, simulation, and augmented reality. Without the traditional complexity associated with 3D reconstruction.
